Common Git Issues
-----------------

| Newcomers to git or distrubuted development may run into git issues frequently.
| Git can be a lot more complicated than a version control system like Subversion.

Here is a list of common problems our community has ran into, and what you can
try to fix them. If your problem isn't listed, search online for a solution.
If you're still stuck, ask on our :ref:`IRC channel <irc-label>` or
:ref:`mailing list <mailing-list-label>` for help.

If you solve your issue, please add your issue and solution to this list!
